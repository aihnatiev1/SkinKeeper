🤝 Community Price Network

SkinKeeper users help each other by sharing price data. When you browse Steam Market, the extension passively reads prices that Steam already shows you — zero extra requests.

This data keeps prices fresh for everyone. No personal data is sent — only item names and prices.

✅ Collect Prices is ON by default

You're helping the community just by browsing Steam. You can turn this off anytime in extension settings. We never make extra requests to Steam — only read what's already loaded.
