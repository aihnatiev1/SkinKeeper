# Plan 14-02: Flutter UX/UI Refactoring — Design, Performance, Architecture

## Goal
Refactor the entire Flutter frontend for speed, elegance, and maintainability. Break god widgets, eliminate unnecessary rebuilds, deduplicate sell/price logic, polish the glassmorphic design system, and deliver a noticeably faster, smoother user experience.

## Context
Flutter app with Riverpod, GoRouter, glassmorphic dark theme. 58+ Dart files. Main issues: `inventory_screen.dart` is 966 lines, GridView rebuilds entire list on selection, sell flow duplicated 3x, bottom sheet patterns repeated, no `const` constructors, BackdropFilter on every frame, portfolio tabs unmount/remount on switch.

### Task 2: Fix State Rebuild Efficiency
**Files:** `lib/features/inventory/inventory_provider.dart`, `lib/features/inventory/widgets/item_card.dart`, `lib/features/portfolio/portfolio_screen.dart`

1. **Selection state optimization:**
   - Replace `StateProvider<Set<String>>` with `StateNotifierProvider` that exposes:
     - `toggle(assetId)` — returns new state only if changed
     - `isSelected(assetId)` — read without watching entire set
     - `selectRange(assetIds)` — batch update, single notification
   - Use `select()` in ItemCard to watch only own selection state:
     ```dart
     final isSelected = ref.watch(
       selectionProvider.select((s) => s.contains(item.assetId))
     );
     ```

2. **Grid item P/L optimization:**
   - Remove `ref.watch(itemPLMapProvider)` from GridView.builder
   - Create `itemPLProvider.family(marketHashName)` — each card watches only its own P/L
   - Cache the P/L map in provider, not recomputed per frame

3. **Portfolio tab optimization:**
   - Replace `AnimatedSwitcher` with `IndexedStack` — keep all tabs mounted
   - Each tab is lazy-loaded on first visit, stays mounted after

4. **filteredInventoryProvider optimization:**
   - Make it a `NotifierProvider` that caches result
   - Only recompute when sort/filter/search inputs actually change
   - Use `Riverpod select()` for granular dependencies

5. **Add `const` constructors** wherever possible across all widgets

**Commit:** `perf(flutter): granular selection state, family providers, IndexedStack tabs`
