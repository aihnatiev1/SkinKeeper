Read `DMARKET_SECRET_KEY` from env. Convert hex to Buffer, take first 32 bytes as seed. Use `crypto.sign(null, Buffer.from(message), { key: seed, format: 'der' })` or tweetnacl `nacl.sign.keyPair.fromSeed(seed)` + `nacl.sign.detached()`. Return hex signature string.

Sign with timestamp. GET `https://api.dmarket.com${fullPath}` with headers `X-Api-Key`, `X-Sign-Date`, `X-Request-Sign: "dmar ed25519 ${signature}"`. Parse `data.objects[0].price.USD` (string cents) -> parseInt / 100. Return null on error.
       - `fetchDMarketPrices(marketHashNames: string[]): Promise<Map<string, number>>` -- Same pattern as CSFloat: deduplicate, check env vars (DMARKET_PUBLIC_KEY + DMARKET_SECRET_KEY), iterate with 200ms delay, collect results.
