# Plan 12-02: Flutter — Integrate Cache with Providers & Offline UI

## Goal
Wire CacheService into inventory/portfolio providers for instant app launch and offline support. Add "last updated" indicator.

## Context
- CacheService (12-01) provides get/put for prices, inventory, portfolio
- `inventoryProvider` in `lib/features/inventory/inventory_provider.dart` fetches via API
- `portfolioProvider` in `lib/features/portfolio/portfolio_provider.dart` fetches via API
- Need: show cached data instantly, fetch fresh in background, show sync indicator

## Tasks

### 1. Update Inventory Provider — Cache-First

File: `lib/features/inventory/inventory_provider.dart`

Strategy: return cached data immediately, then fetch fresh in background.

```dart
@override
Future<List<InventoryItem>> build() async {
  // 1. Try cache first for instant display
  final cached = CacheService.getInventory();
  if (cached != null) {
    // Start background refresh (don't await)
    _refreshInBackground();
    return cached.map((j) => InventoryItem.fromJson(j)).toList();
  }

  // 2. No cache — fetch from API
  return _fetchFromApi();
}

Future<void> _refreshInBackground() async {
  try {
    final fresh = await _fetchFromApi();
    state = AsyncData(fresh);
  } catch (_) {
    // Keep showing cached data on network error
  }
}

Future<List<InventoryItem>> _fetchFromApi() async {
  final response = await ref.read(apiClientProvider).get('/inventory');
  final items = (response.data['items'] as List)
      .map((j) => InventoryItem.fromJson(j as Map<String, dynamic>))
      .toList();
  // Cache the raw JSON for next launch
  CacheService.putInventory(
    (response.data['items'] as List).cast<Map<String, dynamic>>(),
  );
  CacheService.lastSync = DateTime.now();
  return items;
}
```

### 3. Offline Sync Indicator Widget

File: `lib/widgets/sync_indicator.dart` (new)

Small widget for app bar or below it showing "Last updated: 5m ago" or "Offline — showing cached data".

```dart
class SyncIndicator extends ConsumerWidget {
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final label = CacheService.lastSyncLabel;
    final isStale = CacheService.lastSync != null &&
        DateTime.now().difference(CacheService.lastSync!).inHours > 1;

    return Container(
      padding: EdgeInsets.symmetric(horizontal: 12, vertical: 4),
      child: Row(
        mainAxisSize: MainAxisSize.min,
        children: [
          Icon(
            isStale ? Icons.cloud_off : Icons.cloud_done,
            size: 14,
            color: isStale ? Colors.orangeAccent : Colors.white38,
          ),
          SizedBox(width: 4),
          Text(
            label,
            style: TextStyle(fontSize: 11, color: Colors.white38),
          ),
        ],
      ),
    );
  }
}
```

### 6. Handle Network Errors Gracefully

In both providers, catch DioException and fall back to cached data instead of showing error screen:

```dart
try {
  return await _fetchFromApi();
} on DioException {
  final cached = CacheService.getInventory();
  if (cached != null) return cached.map((j) => InventoryItem.fromJson(j)).toList();
  rethrow; // No cache, show error
}
```
