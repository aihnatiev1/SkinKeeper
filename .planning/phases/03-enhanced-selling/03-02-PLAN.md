Create `lib/features/inventory/sell_provider.dart`:

1. `sellOperationProvider` — AsyncNotifierFamily for tracking an active operation:
   - State: SellOperation (operationId, status, items list, succeeded, failed, total)
   - `startOperation(items)`: POST /api/market/sell-operation, then start polling
   - `pollProgress()`: GET /api/market/sell-operation/:id every 1 second
   - `cancelOperation()`: POST /api/market/sell-operation/:id/cancel
   - Auto-stop polling when status is 'completed' or 'cancelled'

2. `sellVolumeProvider` — AsyncNotifierProvider:
   - Fetches GET /api/market/volume
   - Returns { today, limit, warningAt, remaining }
   - `refresh()` method

3. `duplicatesProvider` — FutureProvider:
   - Fetches GET /api/inventory/duplicates
   - Returns list of { marketHashName, count, assetIds, iconUrl, bestPrice }

4. `quickPriceProvider(marketHashName)` — FutureProvider.family:
   - Fetches GET /api/market/quickprice/:marketHashName
   - Returns sellerReceivesCents

5. Fee calculation helper (pure Dart, no API call):
   - `calculateFees(sellerReceivesCents)` returns { buyerPays, steamFee, cs2Fee, sellerReceives }
