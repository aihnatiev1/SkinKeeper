# Plan 10-01: ASO — Localization, Review Prompts, Legal Pages, Metadata

## Goal
Optimize app for App Store / Play Store discoverability: implement full i18n with 6 languages, add smart in-app review prompts, host privacy policy & ToS, fix Android metadata, and prepare store listing copy/keywords.

## Prerequisites
- Phase 9 complete (all features built — screenshots can show full app)
- App runs on iOS and Android simulators

---

## Tasks

### Task 1: Fix Android App Label

**File:** `android/app/src/main/AndroidManifest.xml`

Change `android:label="skin_tracker"` → `android:label="Skin Tracker"`

This matches the iOS `CFBundleDisplayName` ("Skin Tracker") for consistency.

---

### Task 2: Flutter Localization Setup

**2a. Add dependencies to `pubspec.yaml`:**

```yaml
dependencies:
  flutter_localizations:
    sdk: flutter

flutter:
  generate: true
```

The `intl` package is already present (`^0.20.2`).

**2b. Create `l10n.yaml` in project root:**

```yaml
arb-dir: lib/l10n
template-arb-file: app_en.arb
output-localization-file: app_localizations.dart
output-class: AppLocalizations
preferred-supported-locales: [en]
nullable-getter: false
```

### Task 3: In-App Review Prompts

**3a. Add dependency to `pubspec.yaml`:**

**3b. Create review service: `lib/core/review_service.dart`**

```dart
import 'package:in_app_review/in_app_review.dart';
import 'package:shared_preferences/shared_preferences.dart';

class ReviewService {
  static const _lastPromptKey = 'review_last_prompt';
  static const _promptCountKey = 'review_prompt_count';
  static const _maxPrompts = 3;
  static const _cooldownDays = 30;

  static final _inAppReview = InAppReview.instance;

  /// Try to show review prompt after a positive action.
  /// Conditions:
  /// 1. Device supports in-app review
  /// 2. Haven't prompted more than 3 times total
  /// 3. At least 30 days since last prompt
  static Future<void> maybeRequestReview() async {
    try {
      final isAvailable = await _inAppReview.isAvailable();
      if (!isAvailable) return;

      final prefs = await SharedPreferences.getInstance();
      final promptCount = prefs.getInt(_promptCountKey) ?? 0;
      if (promptCount >= _maxPrompts) return;

      final lastPrompt = prefs.getInt(_lastPromptKey) ?? 0;
      final daysSinceLastPrompt =
          DateTime.now().difference(DateTime.fromMillisecondsSinceEpoch(lastPrompt)).inDays;
      if (lastPrompt > 0 && daysSinceLastPrompt < _cooldownDays) return;

      await _inAppReview.requestReview();

      await prefs.setInt(_lastPromptKey, DateTime.now().millisecondsSinceEpoch);
      await prefs.setInt(_promptCountKey, promptCount + 1);
    } catch (_) {
      // Silently fail — review is non-critical
    }
  }
}
```

**3c. Trigger review after positive actions:**

Add calls in these locations:

1. **After successful sell** — in `sell_provider.dart`, after sell completes successfully:
   ```dart
   import '../../../core/review_service.dart';
   // After successful sell completion:
   ReviewService.maybeRequestReview();
   ```

2. **After first profit milestone** — in `portfolio_screen.dart`, when P/L first turns positive:
   ```dart
   // If totalProfit > 0 and user hasn't seen this milestone:
   ReviewService.maybeRequestReview();
   ```

3. **After 5th app open** — in `main.dart`, track app opens via SharedPreferences:
   ```dart
   final prefs = await SharedPreferences.getInstance();
   final opens = (prefs.getInt('app_open_count') ?? 0) + 1;
   await prefs.setInt('app_open_count', opens);
   if (opens == 5) {
     ReviewService.maybeRequestReview();
   }
   ```
